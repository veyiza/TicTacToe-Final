# Tic Tac Toe 

Simple Tic Tac Toe game built with HTML, CSS, and JavaScript. Players take turns placing X and O. The game detects wins, draws, and includes a small scoreboard.

## Features
- Clean 3Ã—3 game board UI with hover + click feedback
- Turn-based gameplay 
- Win detection 
- Draw detection
- Scoreboard 

## Technologies Used
- HTML5
- CSS3 
- JavaScript 

## How to Use
1. Open the app in browser.
2. Click an empty square to place mark.
3. First player to get 3 in a row wins.
4. Press **Reset Game** to clear the board.
